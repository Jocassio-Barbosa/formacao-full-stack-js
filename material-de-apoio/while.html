While no JavaScript

Conceito

  • Estrutura de repetição
  • Executa um bloco de código várias vezes
  • Em inglês, "while" = "enquanto"

Funcionamento

  • Laço que se repete enquanto uma condição for verdadeira
  • Parecido com a estrutura "if"
  • Pode causar loop infinito se a condição nunca se tornar falsa

Sintaxe

while (condição) {  // código a ser repetido  }

Características

  • Aceita o "break" para interromper a repetição
  • Pode ser combinado com outras estruturas como "if" e "switch"

Utilizações

  • Quando precisamos repetir um trecho de código por tempo indeterminado
  • Em programas que precisam ficar executando até o usuário decidir sair
  • Dentro de outras estruturas para criar repetições
