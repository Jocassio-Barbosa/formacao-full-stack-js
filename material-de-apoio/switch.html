A estrutura condicional switch

A aula aborda uma estrutura condicional muito útil em JavaScript chamada switch. O switch é parecido com o if no sentido de que também serve para executar blocos de código baseados em uma condição ou expressão booleana. Porém, o switch tem uma estrutura diferente do if.

O que é o switch?

O switch em inglês significa "trocar". Ele vai testando a expressão contra vários casos, como se fosse trocando entre eles e testando um por um. Quando encontra uma correspondência, executa aquele caso.

É possível definir também um caso padrão que será sempre executado, independente da expressão. E dá para parar a execução a qualquer momento, saindo do switch.

Estrutura básica do switch

A estrutura básica do switch é:

switch(expressão) {  case valor1:     //código a ser executado    break;  case valor2:    //código a ser executado     break;  default:    //código a ser executado}

Analisando a estrutura:

  • switch: palavra chave que define o início do switch.
  • (expressão): expressão ou variável a ser avaliada.
  • case valor: verifica se a expressão é igual ao valor.
  • break: para a execução e sai do switch.
  • default: caso executado se nenhum case corresponder.

O case é como se fosse o "if" dentro do switch. Cada case verifica se a expressão corresponde ao seu valor. Quando encontra, executa aquele código interno.

O break é importante para parar a execução depois que um case foi correspondido. Senão, ele continuará executando os próximos casos mesmo que não correspondam.

O default funciona como o "else" do if. É executado caso nenhum case tenha sido verdadeiro.

Exemplo prático de switch

Vamos ver um exemplo prático de switch no JavaScript:

let resultado = "B";switch (resultado) {  case "A":    alert("O resultado é A");     break;    case "B":    alert("O resultado é B");    break;  case "C":     alert("O resultado é C");    break;    default:     alert("Finalizando");}

Aqui definimos uma variável resultado com valor "B". Este valor é passado para a expressão do switch.

Então ele vai verificar caso a caso:

  1. Case "A" não corresponde
  2. Case "B" corresponde. Executa o alerta "O resultado é B"
  3. Encontra o break, então para a execução e sai.

Perceba que sem os breaks, ele iria executar os próximos casos mesmo sem corresponderem.

O default também seria executado ao final.

Usando switch com interação

O switch permite avaliar não só valores fixos, mas qualquer tipo de expressão ou variável. Isso permite criar interações com usuário.

Exemplo:

let resultado = prompt("Escolha uma alternativa: \nA\nB\nC"); switch (resultado) {  case "A":    alert("Resultado é A");     break;    case "B":     alert("Resultado é B");    break;  case "C":    alert("Resultado é C");    break;  }

Aqui usamos prompt() para pedir uma entrada do usuário e armazenamos na variável resultado.

Essa entrada é então avaliada no switch para exibir a mensagem correspondente.

Usando números no switch

Os cases não precisam ser apenas strings. Podemos avaliar números:

let num = parseInt(prompt("Digite um número"));switch (num) {  case 1:    alert("Número 1");    break;  case 2:    alert("Número 2");     break;    case 3:    alert("Número 3");    break;  default:    alert("Número não identificado");  }

Usamos parseInt() para converter a entrada para número e depois comparamos com os cases numéricos.

Assim testamos valores numéricos facilmente.

Vantagens do switch

O switch tem algumas vantagens em relação uma cadeia de ifs/elses quando queremos testar múltiplos valores:

  • Código mais limpo e fácil de ler
  • Execução mais rápida em alguns casos
  • Permite definir um caso padrão
  • Podemos sair facilmente da estrutura com break

Então o switch é muito útil quando temos uma variável que assumirá poucos valores distintos.

Conclusão

Em resumo, o switch é uma estrutura condicional poderosa para avaliar uma expressão/variável contra múltiplos casos.

Sua estrutura permite definir casos e um padrão de forma fácil.

Por poder parar a execução com break, o switch economiza processamento desnecessário.

Dessa forma, o switch é perfeito para situações em que se precisa tomar decisões com variáveis que assumem um conjunto limitado de valores.
