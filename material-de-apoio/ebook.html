Arrays no JavaScript

Introdução aos Arrays

Os Arrays são uma estrutura de dados importante no JavaScript que permite armazenar e manipular coleções de elementos.

Um Array pode conter elementos de qualquer tipo, como números, strings, boleanos, objetos e até mesmo outros Arrays. Cada elemento é identificado por um índice numérico que começa em 0. 

// Exemplo de Array com elementos de tipos diferentesconst meuArray = [5, "Maria", true, {nome: "João"}, [1, 2]];

Os principais benefícios de usar Arrays são:

  • Armazenar múltiplos valores em uma única variável
  • Acessar, iterar e manipular os elementos facilmente por meio do índice
  • Adicionar e remover elementos dinamicamente, alterando o tamanho do Array

Veremos agora com mais detalhes como criar, acessar elementos e realizar diversas operações com Arrays no JavaScript.

Criando Arrays

Há duas formas principais de criar um Array:

1. Utilizando Colchetes []

Para criar um Array vazio, basta declarar uma variável e atribuir a ela um par de colchetes []: 

const arr = [];

Também é possível já inicializar o Array com alguns elementos, separando-os por vírgula dentro dos colchetes: 

const numeros = [5, 10, 15, 20];

2. Utilizando o Construtor Array()

Outra forma é utilizando o construtor Array(), passando como parâmetro o tamanho do Array: 

const nomes = new Array(3); // cria Array de tamanho 3

Ou já inicializando com valores:

const frutas = new Array("Maçã", "Banana", "Laranja");

A forma mais comum é utilizando colchetes [], por ser mais simples e direta.

Acessando Elementos

Para acessar um elemento específico de um Array, utilizamos a notação de colchetes [], passando dentro deles o índice numérico desejado, lembrando que o primeiro elemento está no índice 0: 

const cores = ["vermelho", "verde", "azul"]; cores[0]; // retorna "vermelho" cores[1]; // retorna "verde"cores[2]; // retorna "azul"

Caso o índice não exista, será retornado undefined.

Também é possível modificar um elemento do Array utilizando a notação de colchetes para acessá-lo e o operador de atribuição (=) para alterar seu valor: 

let numeros = [5, 10, 15];numeros[1] = 20; // agora numeros = [5, 20, 15]

Propriedades e Métodos

Os Arrays possuem propriedades e métodos próprios que permitem inspecionar e manipular os elementos. Alguns dos mais utilizados são:

length - retorna o tamanho do Array, ou seja, número de elementos: 

const carros = ["Ferrari", "Fusca", "Palio"];carros.length; // 3

push() - adiciona novo elemento ao final do Array:

const alunos = ["João", "Maria"];alunos.push("José"); // agora alunos = ["João", "Maria", "José"]

pop() - remove o último elemento do Array:

const frutas = ["Banana", "Maçã", "Uva"];frutas.pop();// frutas = ["Banana", "Maçã"]

unshift() - adiciona elemento ao início do Array:

const num = [2, 3, 4];num.unshift(1);// agora num = [1, 2, 3, 4]

shift() - remove o primeiro elemento do Array:

const num = [5, 2, 6]; num.shift();// agora num = [2, 6]

E muitos outros! Consulte a documentação para ver todos os métodos disponíveis.

Iterando em Arrays

Podemos percorrer todos os elementos de um Array utilizando laços de repetição, como o for: 

const frutas = ["Maçã", "Banana", "Laranja"];for(let i = 0; i < frutas.length; i++) {  console.log(frutas[i]); }// imprime cada fruta no console

Outra forma é utilizando o método forEach():

frutas.forEach(function(fruta) {  console.log(fruta)  });// também imprime cada fruta no console

O forEach() recebe uma função callback que é executada para cada elemento.

Existem diversas formas de iterar Arrays, cada uma com suas vantagens.

Verificando se é Array

Podemos verificar se uma variável é um Array utilizando o método Array.isArray(): 

const carros = ["Ferrari", "Fusca", "Palio"];Array.isArray(carros); // trueArray.isArray({nome: "João"}); // false

Retorna true caso seja um Array, e false para qualquer outro tipo.

Convertendo em String

Para converter um Array em String, basta utilizar o método join(), que recebe um separador como parâmetro: 

const frutas = ["Banana", "Maçã", "Laranja"];frutas.join(); // Banana,Maçã,Laranja frutas.join(" | "); // Banana | Maçã | Laranja

Por padrão o separador é vírgula, mas pode ser definido.

Ordenando Arrays

Podemos ordenar os elementos de um Array com o método sort():

const numbers = [4, 2, 8, 1];numbers.sort(); // numbers agora é [1, 2, 4, 8]

Por padrão ele ordena os elementos alfabeticamente. Para ordenar números, deve-se passar uma função callback de comparação: 

numbers.sort(function(a, b) {  return a - b; });// agora está ordenado numericamente

Existem diversas formas de ordenação que podem ser implementadas.

Buscando elementos

Alguns métodos úteis para buscar elementos são:

indexOf() - retorna o primeiro índice em que um elemento é encontrado: 

const frutas = ["Banana", "Caju", "Banana" ] frutas.indexOf('Banana') // 0

lastIndexOf() - retorna o último índice do elemento:

frutas.lastIndexOf('Banana') // 2

find() - retorna o primeiro elemento que satisfaz a condição: 

const numbers = [4, 9, 16, 25, 29];numbers.find(x => x > 20); // 25

findIndex() - retorna o índice do primeiro elemento que satisfaz a condição: 

numbers.findIndex(x => x > 20); // 3

includes() - retorna true se o array possuir o elemento:

numbers.includes(25); // true

Estes são alguns métodos que permitem buscar elementos de forma prática.

Removendo elementos

Alguns métodos para remoção de elementos:

pop() / shift() - removem o último e o primeiro elemento respectivamente

splice() - remove um ou mais elementos em qualquer posição: 

const nomes = ["Maria", "José", "João"]// Remove José (índice 1)nomes.splice(1, 1); // nomes = ["Maria", "João"]

O primeiro parâmetro é o índice inicial, e o segundo quantos elementos remover a partir dele.

filter() - retorna um novo array somente com elementos que passam em uma condição: 

const numbers = [4, 9, 16, 25, 29];const maiores20 = numbers.filter(n => n > 20); // agora maiores20 é [25, 29]

Existem outras formas como slice() e concat() que também removem elementos.

Concatenando Arrays

Podemos concatenar dois Arrays com o método concat():

const frutas1 = ["Banana", "Maçã"];const frutas2 = ["Uva", "Laranja"];const todasFrutas = frutas1.concat(frutas2);// todasFrutas = ["Banana", "Maçã", "Uva", "Laranja"]

Isso permite unir dois Arrays de forma simples.

Fazendo cópias de Arrays

Devido a características do JavaScript, ao atribuir um Array para outra variável, ambas apontam para o mesmo local na memória.

Para criar de fato uma cópia, deve-se manualmente copiar os elementos, ou utilizar métodos como slice() e concat(): 

const arr1 = [1, 2, 3];const arr2 = arr1.slice(); arr2.push(4); // arr1 permanece [1, 2, 3]// arr2 agora é [1, 2, 3, 4]

O slice() sem parâmetros faz uma cópia completa do Array.

String vs Array

Uma string pode parecer um Array mas não possui as mesmas funcionalidades. Uma string é uma sequência de caracteres, enquanto um Array é uma coleção de valores.

Strings possuem o método split() que divide a string em um Array:

"a,b,c".split(","); // ["a", "b", "c"]

Já Arrays possuem o método join() que converte para string.

Conclusão

Vimos uma introdução abrangente sobre Arrays no JavaScript, uma estrutura flexível e indispensável na linguagem.

Aprendemos desde a criação e manipulação básica, até métodos avançados de busca, ordenação e transformação de Arrays.

Há muito mais a se explorar, mas com estes conceitos é possível resolver uma grande variedade de problemas envolvendo conjuntos de dados.

Os Arrays são estruturas versáteis presentes em todas as linguagens de programação e que todo desenvolvedor deve conhecer profundamente.

Métodos de Array em JavaScript

Introdução

Nesta aula, vamos conhecer quatro métodos muito úteis para trabalhar com arrays em JavaScript: map, filter, reduce e sort.

Esses métodos são chamados de higher-order functions, ou seja, funções de primeira classe que recebem outras funções como argumentos ou retornam outras funções.

Vamos entender como cada um desses métodos funciona e ver exemplos práticos de uso.

Map

O método map() é utilizado para mapear cada elemento de um array para um novo elemento e retornar um novo array com os elementos mapeados.

Ou seja, o map() aplica uma função de callback provida para cada elemento do array original e retorna um novo array com os resultados. A função callback é executada para cada elemento durante a iteração.

A sintaxe básica do map() é:

const novoArray = arr.map(callback)

Onde:

  • arr é o array original que será mapeado
  • callback é a função que será executada em cada elemento
  • novoArray é o novo array contendo os elementos mapeados

Vamos ver um exemplo prático:

const numeros = [2, 4, 6, 8]const dobro = numeros.map(n => n * 2) console.log(dobro) // [4, 8, 12, 16]

No exemplo acima, passamos uma arrow function que retorna o dobro de cada elemento. Dessa forma, o map() percorreu o array original, aplicou a função em cada elemento e retornou um novo array com os valores dobrados.

É importante notar que:

  • O map() não altera o array original, apenas retorna um novo
  • A função de callback recebe 3 parâmetros: o elemento, o índice e o array
  • A função deve retornar o novo elemento mapeado

Outro exemplo aplicando map():

const nomes = ['Ana', 'Bia', 'Gui']const nomesMaiusculos = nomes.map(nome => nome.toUpperCase())console.log(nomesMaiusculos) // ['ANA', 'BIA', 'GUI']

Novamente transformamos cada elemento do array (nome) em maiúsculo e retornamos um novo array com os nomes maiusculizados.

O map() é muito útil para converter dados de um formato para outro.

Filter

O método filter() é utilizado para filtrar um array, retornando um novo array apenas com os elementos que passaram no teste implementado pela função callback provida.

A sintaxe básica é:

const novoArray = arr.filter(callback)

Onde:

  • arr é o array original a ser filtrado
  • callback é a função que testa cada elemento
  • novoArray retorna os elementos filtrados

Vamos a um exemplo:

const idades = [22, 8, 17, 14, 65]const maiores = idades.filter(idade => idade >= 18)console.log(maiores) // [22, 65]

No exemplo acima, filtramos as idades para retornar um array apenas com os maiores de 18 anos.

O filter é muito útil para filtrar arrays e retornar dados que atendem a certos critérios.

Outro exemplo:

const produtos = [  { nome: 'Camisa', preco: 25 },  { nome: 'Bermuda', preco: 50 },  { nome: 'Casaco', preco: 120 }  ]const caros = produtos.filter(p => p.preco >= 50) console.log(caros) // [{ nome: 'Bermuda', preco: 50 }, { nome: 'Casaco', preco: 120 }]

Aqui filtramos os produtos com preço maior ou igual a 50 em um novo array.

Reduce

O método reduce() é utilizado para reduzir o array a um único valor.

Ele aplica uma função callback aos elementos do array, resultando num único valor de retorno.

A sintaxe básica é:

const valor = arr.reduce(callback)

Onde:

  • arr é o array a ser reduzido
  • callback é a função que será executada em cada elemento
  • valor é o valor final reduzido

Vamos a um exemplo de soma de números:

const numbers = [15, 25, 35, 45]const sum = numbers.reduce((acc, cur) => acc + cur)console.log(sum) // 120

No exemplo acima, a função callback recebe dois parâmetros:

  • acc: o acumulador, que guarda o valor total
  • cur: o elemento atual sendo processado

No primeiro loop, acc começa com valor 15 (primeiro elemento) e cur vale 25 (segundo elemento). São somados resultando em 40.

No segundo loop, acc vale 40 e cur vale 35. Somados resultam em 75.

E assim por diante, até retornar o valor final reduzido de 120.

Outro exemplo de uso do reduce():

const carrinho = [  { nome: 'Notebook', qtd: 2, preco: 2500 },  { nome: 'Smartphone', qtd: 1, preco: 1500 }]const total = carrinho.reduce((acc, cur) => {  return acc + cur.qtd * cur.preco}, 0)console.log(total) // 5500

Neste caso, calculamos o valor total do carrinho de compras somando todos os subtotais.

O reduce() é muito poderoso para cálculos e redução de arrays!

Sort

O método sort() ordena os elementos do próprio array em que foi chamado e retorna o array. A ordenação padrão é ascendente.

A sintaxe básica é:

arr.sort()

Contudo, o sort tem um comportamento interessante: por padrão, ele ordena strings de acordo com a posição na tabela Unicode.

Para ordenar strings de forma mais intuitiva ou ordenar outros tipos de dados, precisamos passar uma função callback que compara os elementos.

Exemplo com strings:

const frutas = ['Banana', 'Amora', 'abacaxi', 'uva'];frutas.sort(); console.log(frutas); // ['Amora', 'Banana', 'abacaxi','uva']

Perceba que ordenou de forma errada por causa do algoritmo interno. Para consertar: 

frutas.sort((a, b) => {  if (a > b) return 1    if (b > a) return -1    return 0})console.log(frutas) // ['abacaxi', 'Amora', 'Banana', 'uva']

Agora sim ordenou corretamente!

Exemplo ordenando números:

const numbers = [4, 2, 6, 1]  numbers.sort((a, b) => a - b)console.log(numbers) // [1, 2, 4, 6]

E para inverter a ordem:

numbers.sort((a, b) => b - a) console.log(numbers) // [6, 4, 2, 1]

Algumas observações importantes sobre o sort():

  • Modifica o array original (cuidado!)
  • Por padrão, ordena strings por Unicode
  • Aceita uma função callback para comparação customizada
  • Retorna o array ordenado

Uma dica é usar .slice() antes para não modificar o array original: 

const numbers = [5, 1, 4, 2, 3]const sorted = numbers.slice().sort((a, b) => a - b)console.log(numbers) // array original não foi modificado console.log(sorted) // novo array ordenado

Assim conseguimos ordenar sem afetar o array original, o que pode ser útil.

Conclusão

Esses são 4 métodos de array muito poderosos do JavaScript que todo desenvolvedor precisa conhecer.

O map(), filter(), reduce() e sort() são high-order functions que nos permitem fazer operações avançadas e complexas em arrays de forma muito simples.

Dominar o uso desses métodos vai te ajudar a escrever códigos JavaScript mais elegantes, concisos e declarativos para lidar com dados em arrays.

Eles nos permitem evitar loops tradicionais com for/while na maioria dos casos.

Portanto, pratique bastante os exemplos para incorporar o uso desses métodos no seu dia-a-dia. Isso vai melhorar suas habilidades no JavaScript moderno.

Bons estudos!

Objetos Globais do JavaScript

Introdução aos Objetos Globais

Os objetos globais do JavaScript nada mais são do que objetos padrão da linguagem que ficam sempre disponíveis para serem utilizados no código.

Eles funcionam de forma similar às estruturas como condicionais e repetições (if, switch, while, for), que também fazem parte da linguagem JavaScript e podem ser referenciadas diretamente no código sem a necessidade de serem importadas ou definidas anteriormente.

A principal diferença é que os objetos globais não são estruturas, e sim objetos no sentido mais literal, possuindo propriedades e métodos como qualquer outro objeto JavaScript.

Diferença entre Objetos Globais e o Objeto Window

É importante não confundir os objetos globais com o objeto de escopo global window do navegador ou o escopo global de qualquer outro ambiente JavaScript.

O objeto window representa o escopo global no contexto do navegador. Por isso, tudo que está disponível como propriedade ou método do window pode ser acessado diretamente no código dentro do navegador, sem a necessidade de referenciar o próprio objeto window.

Já os objetos globais fazem parte apenas da linguagem JavaScript, então podem ser utilizados independente do ambiente.

Principais Objetos Globais

Alguns dos principais objetos globais do JavaScript incluem:

  • Object - permite criar novos objetos JavaScript;
  • Array - usado para criar e manipular arrays;
  • String - para trabalhar com strings;
  • Number - para trabalhar com números;
  • Math - contém propriedades e métodos para realizar operações matemáticas;
  • Date - para manipulação de datas;

Essas são apenas algumas das dezenas de objetos globais disponíveis.

Algumas características desses objetos:

  • Nomes sempre começam com letra maiúscula;
  • Representam tipos de dados internos do JavaScript;
  • Possuem propriedades e métodos úteis.

Usos Comuns dos Objetos Globais

Vejamos alguns exemplos práticos de uso desses objetos globais:

Object

O objeto global Object permite criar novos objetos JavaScript.

const person = Object(); // cria um objeto vazioconst list = Object([1, 2, 3]); // cria um array preenchido

Array

O Array global possui a mesma utilidade, mas específico para arrays:

const list = Array(); // array vazioconst names = Array("Ana", "Bia", "Carlos"); // array preenchido

String e Number

String e Number convertem valores informados para seus respectivos tipos:

String(42); // retorna a string "42"Number("42"); // retorna o número 42

Também é possível criar strings e números a partir desses objetos.

Math

O objeto Math contém várias propriedades e métodos para realizar operações matemáticas sem a necessidade de implementá-las manualmente: 

Math.PI // retorna 3.14159Math.pow(2, 3); // retorna 8 Math.sqrt(144); // retorna 12

Date

O objeto Date permite trabalhar com datas de forma simples:

Date(); // retorna a data e hora atual

Conclusão

Os objetos globais JavaScript provêm funcionalidades úteis que seriam mais complexas de implementar manualmente.

Conhecer os objetos globais disponíveis e suas aplicações permite aproveitá-los para facilitar e agilizar o desenvolvimento.

Eles não precisam ser importados ou definidos anteriormente para serem utilizados, estando sempre disponíveis globalmente.

Consultar a documentação da linguagem para conhecer mais objetos globais pode ajudar a potencializar ainda mais os códigos JavaScript.
